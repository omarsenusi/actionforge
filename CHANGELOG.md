# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-07-04

### Added
- Core detectors for package manager, frameworks (NestJS, AdonisJS, Next.js, Express, Fastify, Node.js), and Node version.
- Handlebars templates for automatic GitHub Actions CI generation.
- Interactive CLI setup wizard `actionforge init`.
- Non-interactive flag-based generator command `actionforge generate`.
- Troubleshooter diagnostic check `actionforge doctor`.
