# ActionForge Project Roadmap

This roadmap outlines planned features, framework support, and pipeline expansions.

## Phase 1: Core Frameworks (Completed)
- [x] Node.js (Generic)
- [x] NestJS
- [x] AdonisJS
- [x] Next.js
- [x] Express
- [x] Fastify

## Phase 2: Extended Frameworks & Tools
- [ ] Nuxt.js
- [ ] Astro
- [ ] Remix
- [ ] SvelteKit
- [ ] Laravel (PHP)
- [ ] Django / FastAPI (Python)

## Phase 3: Docker & Containerization
- [ ] Auto-detection of Dockerfiles
- [ ] Generation of multi-arch Docker build & push workflows (GitHub Packages / DockerHub)
- [ ] Kubernetes manifest validations and linting

## Phase 4: Multiple Target Engines
- [ ] GitLab CI/CD YAML generator
- [ ] CircleCI Config generator
- [ ] Azure DevOps Pipelines YAML generator
- [ ] Bitbucket Pipelines generator
